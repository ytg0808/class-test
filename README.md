# ☀️ 아침 독서 요정 | 초등 온라인 독서기록장

초등학교 5학년 학생들이 아침 자율 독서 시간에 쉽고 재미있게 독서량을 기록하고, 실시간 랭킹 보드를 통해 독서 습관을 재미있게 들일 수 있도록 디자인된 반응형 웹앱입니다.

이 프로젝트는 **Next.js 14 (App Router) + TypeScript + Tailwind CSS**로 구성되어 있으며, 데이터는 로컬 스토리지(`LocalStorage`)를 기반으로 영속화되므로 서버나 외부 데이터베이스 구축 없이 브라우저에서 즉시 데이터가 유지됩니다. (추후 DB 연동이 용이하게끔 API 및 스토리지 접근 코드가 유틸리티로 추상화되어 있습니다.)

---

## 🚀 시작 가이드 (Local Run)

프로젝트를 내 컴퓨터에서 실행하려면 Node.js 환경이 필요합니다. 아래 절차에 따라 실행해 주세요.

### 1. Node.js 설치
- [Node.js 공식 홈페이지](https://nodejs.org)에 접속하여 최신 LTS 버전을 다운로드하고 설치해 주세요. (설치가 완료되면 `npm`과 `npx` 명령어가 사용 가능해집니다.)

### 2. 의존성 패키지 설치
프로젝트 루트 폴더(`class-test`)에서 터미널(PowerShell 또는 CMD)을 열고 다음 명령어를 입력하세요.
```bash
npm install
```

### 3. 로컬 개발 서버 실행
설치가 끝나면 아래 명령어로 아침 독서 요정 서버를 켭니다.
```bash
npm run dev
```

### 4. 접속하기
서버가 켜지면 크롬 등 인터넷 브라우저를 열고 아래 주소로 접속합니다.
- 🔗 **접속 주소:** [http://localhost:3000](http://localhost:3000)

## ✨ 주요 제공 기능

### 1. ✍️ 학생용 독서기록 입력
- **내 이름 고르기:** 5학년 학생들이 텍스트를 오타 내지 않도록, 귀여운 아바타 이모지가 그려진 이름 선택창을 제공합니다. (기본 가상 학생 15명 제공)
- **책 제목 자동완성:** 한 번이라도 반 학생들이 입력한 도서명이면, 몇 글자 타이핑 시 밑에 자동 완성 목록으로 표시되어 오타를 줄여줍니다.
- **실시간 쪽수 자동 계산:** 시작 쪽수와 끝 쪽수를 적으면 "오늘 총 OO쪽 읽었어요!"라고 실시간 계산되어 재미와 성취감을 줍니다.
- **⚠️ 쪽수를 모르겠어요 기능:** 쪽수를 깜빡 잊었을 때 체크박스를 누르면, 임시로 대략 20쪽 읽은 것으로 임시 저장되며 선생님이 확인 후 승인하는 '교사 대기 상태'로 저장됩니다.
- **한 줄 느낀 점:** 5학년 학생들의 눈높이에 맞는 힌트 플레이스홀더를 제공하며, 등록 성공 시 화면에 화려한 폭죽(Confetti) 효과가 펼쳐집니다!

### 2. 🏆 실시간 독서 랭킹 보드 (Gamification)
- **이번 주 독서 왕:** 이번 주(월요일~일요일) 가장 많이 읽은 학생 1~3등을 시상대(Podium) 그래픽으로 예쁘게 돋보이게 합니다. (1등은 머리에 황금 왕관 👑)
- **월간/주간 랭킹 리스트:** 4위 이하 모든 학생의 데이터도 예쁜 칭찬 스티커 디자인 카드와 함께 탭 정렬(쪽수 기준, 권수 기준)하여 표시합니다.
- **나의 독서 챌린지:** 개인별 목표(월 5권 / 500페이지) 대비 현재 달성도를 실시간 프로그레스 바와 달성률(%)로 시각화해 줍니다.

### 3. 👩‍🏫 선생님 관리판 (Teacher Admin Panel)
- **비밀번호 잠금:** 학생들의 오작동을 막기 위해 잠금 화면을 제공합니다. (초기 비밀번호: `5555`)
- **오늘 아침 독서 출석부:** 오늘 아침에 독서기록장을 등록한 학생(완료 시간 초록색 표시)과 등록하지 않은 학생(회색 미작성 표시)을 출석부 바둑판 형태로 한눈에 보여줍니다.
- **쪽수 대기 승인 관리:** 학생들이 '쪽수를 모르겠어요'로 임시 저장한 기록들을 선생님이 직접 확인 후, 실제 쪽수를 수정해 주거나 클릭 한 번으로 일괄 승인할 수 있는 에디터 모달을 탑재하고 있습니다.
- **엑셀 다운로드 (CSV 추출):** 버튼 클릭 한 번으로 모든 기록 데이터를 한글 깨짐 없이 UTF-8 CSV 파일로 내려받아 엑셀로 활용할 수 있습니다.
